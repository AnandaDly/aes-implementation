# Implementasi RSA Sederhana dengan Streamlit

Aplikasi ini adalah implementasi sederhana dari algoritma enkripsi dan dekripsi RSA menggunakan **Streamlit** sebagai antarmuka pengguna. RSA adalah algoritma enkripsi yang menggunakan kunci publik dan kunci privat untuk mengamankan pesan.

## Fitur

- **Generate Kunci RSA**: Menghasilkan pasangan kunci publik dan kunci privat berdasarkan dua bilangan prima.
- **Enkripsi Pesan**: Menggunakan kunci publik untuk mengenkripsi pesan yang dimasukkan oleh pengguna.
- **Dekripsi Pesan**: Menggunakan kunci privat untuk mendekripsi pesan terenkripsi kembali ke pesan asli.
- **Antarmuka Sederhana**: Antarmuka aplikasi yang intuitif dengan **Streamlit**.
